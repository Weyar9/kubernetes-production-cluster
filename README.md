# Kubernetes Production Cluster

This project demonstrates deploying containerized applications on a Kubernetes cluster.

## Overview

A simple cloud infrastructure project that deploys containerized microservices using Kubernetes and Docker.

## Technologies

Kubernetes
Docker
Linux
Cloud Infrastructure

## Features

Containerized applications
Kubernetes deployments
Rolling updates
Service scaling

## Architecture

Docker containers are deployed to a Kubernetes cluster where services are automatically scaled and managed.

## Goal

Demonstrate cloud platform engineering skills including container orchestration, scalable deployment, and modern infrastructure practices.
